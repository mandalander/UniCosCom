# **App Name**: App Navigator

## Core Features:

- Top Navigation Bar: Display a top navigation bar with a login button on the right and a sidebar toggle button on the left.
- Sidebar Menu: Implement a slide-out sidebar menu that can be toggled using the button in the top navigation bar.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to provide a professional look and feel.
- Background color: Light gray (#EEEEEE) for a clean and modern backdrop.
- Accent color: Teal (#009688) for interactive elements to ensure they stand out.
- Font: 'Inter', a grotesque-style sans-serif font that will be used throughout the whole website.
- Use a responsive layout to adapt to different screen sizes.
- Use simple and clear icons for the sidebar menu items and the toggle button.
- Implement smooth transitions for the sidebar menu opening and closing.
